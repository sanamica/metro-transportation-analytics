# EV Infrastructure & Transportation Analytics

## Project Overview
This project analyzes transportation budget allocation, EV charging infrastructure, and demographic trends using Power BI and Python.

The platform integrates:
- Metro transportation budget data
- EV charging station datasets
- Census API demographic data
- Forecasting models
- Anomaly detection models

## Technologies Used
- Power BI
- Python
- pandas
- scikit-learn
- Census API
- GitHub

## Key Features
- Transportation budget variance analysis
- EV charging accessibility analysis
- Infrastructure gap detection
- Geographic heatmaps
- Forecasting dashboard
- Anomaly detection
